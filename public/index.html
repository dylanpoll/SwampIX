Welcome to ROGLE

A rogue-like game with content generated by ML! Including the name!


Kudos to the following

Monstername API,
Used to generate funny names. We blended results to make them more obscure and added a descriptor class. This occurs as a first step in the process and the name created is used later in the ML image generation.
https://monsternames-api.com/endpoints
OpenAI,
Used to generate the card art for each card, this is generated from the names generated prior.
https://openai.com/api/
Appwrite,
We are using this to run our backend as an alternative to firebase that can be self hosted to use a realtimeDB system.
https://appwrite.io/docs/realtime
